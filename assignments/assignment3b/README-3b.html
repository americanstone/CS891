Assignment Structure and Functionality

The ImageCrawler app is packaged as a project using the latest version of Android Studio. This app is written in both Kotlin and Java and demonstrates many Android capabilities. For the purposes of Assignment 3a, however, you only need to be aware of the following three directories:

  • image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils, which contains the skeleton(s) you'll need to fill in, as described below.
  • app/src/test, which is a set of JUnit tests that exercise the features under test and can be used to evaluate whether you've correctly satisfied this assignment's requirements.
  • app/src/androidTest/java/assignmentTests/assignment2cTests, which is an Android Studio instrumentation test that runs your app automatically.

To compile this code you need to use the provided Android Studio project. You can run this project by clicking the green "Run 'app'" button in the Android Studio IDE, which should automatically select an Android Emulator to run, assuming you have one created. The application's minimum API is 26, so you will need to select an emulator that uses an API >= 26. If you do not already have one created you can create it by clicking on the "AVD Manager" button in the Android Studio IDE.

Program Description and "TODO" Tasks

In this part of the assignment you'll complete the implementation of an image crawler class in the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/crawlers/ForkJoinCrawler2.java file that obtains, stores, and transforms images. This class uses Java fork-join pool framework to perform an "image crawl" starting from a root Uri. Depending on the parameters used to run the tests, images from HTML page reachable from the root Uri are either downloaded from a remote web server or read from the local file system and the results are stored in files.

All students must use Java lambda expressions, method references, and sequential streams for their ForkJoinCrawler2.java implementation. In addition, all students must implement and use the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils/SynchronizedArray.java file instead of the image-crawler/src/main/java/edu/vanderbilt/imagecrawler/utils/UnsynchronizedArray.java.

JUnit tests in the image-crawler/src/test/java/edu/vanderbilt/imagecrawler folder are provided to increase our confidence that your implementation is working as expected.
